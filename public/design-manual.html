Poplist Design System

あそびのあるミニマリスト · Playful Minimalist

Philosophy

Tone: Minimal, calm, and bold — with a hint of humour.

Aesthetic: Clean typography, generous whitespace, but with personality. Nothing feels corporate. Everything feels intentional.

Accessibility: Large readable text. High contrast. Simple, scannable layouts.

Theming: The accent colour changes per user preference. Everything else stays neutral.

Usage Guidelines

Typography:

Body text is --text-base (14px). Never 13px, 15px, or 16px for body. Keep it consistent.

Spacing:

All padding, margin, and gaps use the spacing scale. Never arbitrary pixel values.

Colours:

Always use semantic tokens (--color-text-1), never primitives (--gray-900). This enables seamless theming.

Border Radius:

Hierarchy: smaller elements get smaller radius. Badges use --radius-xs, cards use --radius-lg.

Transitions:

Use var(--transition) for standard interactions. Use --duration-fast for micro-interactions, --duration-slow for modals.
