# Poplist

Voice-first task list app. Speak your tasks — AI classifies them automatically.

## Pages

| File | Purpose |
|---|---|
| `index.html` | Full app (main working file) |
| `welcome.html` | First-open screen mock |
| `app.html` | Task list screen mock |
| `settings-mock.html` | Settings panel mock |

## Deploy

Hosted on Cloudflare Pages — push to `main` branch to auto-deploy.

## Tech

Plain HTML / CSS / JS — no build step required.
Voice recognition via Web Speech API (Chrome / Safari on iOS 17+).
